## Сайт подбора новостроек

Небольшой статический сайт (HTML/CSS/JS) для подбора новостроек по бюджету.

### Как запустить

- **Самый простой способ**: откройте `index.html` двойным кликом в браузере.
- **Рекомендуется** (чтобы корректно работали относительные пути): запустить простой локальный сервер:
  - Windows PowerShell:

```bash
cd "c:\Users\Work\Desktop\Проекты Курсор\Тест"
python -m http.server 5500
```

После этого откройте в браузере `http://localhost:5500`.

### Страницы

- `index.html` — приветственный лендинг.
- `Подбор ЖК.html` — страница подбора ЖК по бюджету.

### Файлы

- `assets/styles.css` — общие стили и токены темы (светлая/тёмная).
- `js/theme.js` — переключение темы с сохранением в `localStorage`.
- `js/app.js` — логика подбора и вывод карточек.

### Тема (светлая/тёмная)

- Кнопка переключения — в правом верхнем углу.
- Выбор сохраняется в `localStorage` (ключ `jk-theme`).
